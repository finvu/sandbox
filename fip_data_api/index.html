    FIP Data API

    This API can be used to add sample data for accounts and associated transactions and can be called from an appropriate testing tool such as Postman

    Adding User Account
    URL http://api.finvu.in/Accounts/add
    HTTP METHOD POST

    Request Body

    <UserAccountInfo>
    <UserAccount accountRefNo="RefHDFC0001" accountNo="HDFC11111111" accountTypeEnum="SAVINGS" FIType="DEPOSIT">
      <Identifiers pan="AFYPA24444" mobile="7030608902" email="info@cookiejar.co.in" aadhar="679486011111"></Identifiers> 
    </UserAccount>
    <UserAccount accountRefNo="RefHDFC0002" accountNo="HDFC22222222" accountTypeEnum="CURRENT" FIType="DEPOSIT">
      <Identifiers pan="AFYPA24444" mobile="7030608902" email="info@cookiejar.co.in" aadhar="679486011111"></Identifiers> 
    </UserAccount>
    <UserAccount accountRefNo="RefSBI0001" accountNo="SBI11111111" accountTypeEnum="SAVINGS" FIType="DEPOSIT">
      <Identifiers pan="AFYPA24444" mobile="7030608902" email="info@cookiejar.co.in" aadhar="679486011111"></Identifiers> 
    </UserAccount>
</UserAccountInfo>

    Response Expected: 

    User account(s) added.

  Account Reference Number : RefHDFC0001
  Account Number : HDFC11111111
  Consent Id : 2c84eba9-4637-45f5-b42e-00e67c128258
 (It use in Account Discovery.)


  Account Reference Number : RefHDFC0002
  Account Number : HDFC22222222
  Consent Id : 09670e7c-7dbb-4d45-8e15-dfcc829caab6
 (It use in Account Discovery.)


  Account Reference Number : RefSBI0001
  Account Number : SBI11111111
  Consent Id : b583190a-d51e-4ce2-9981-f51d0e010af9
 (It use in Account Discovery.)

 Please save this information for FI request API calls.

    Adding User Account Transaction
    URL http://api.finvu.in/Accounts/Transaction/add
    HTTP METHOD POST

    Request Body

    <UserAccountTrans>
        <UserAccount> 
            <accountRefNo>RefHDFC0001</accountRefNo> 
            <accountNo>HDFC11111111</accountNo> 
            <FIType>DEPOSIT</FIType>
            <accountTypeEnum>SAVINGS</accountTypeEnum>
            <accountData>
    <![CDATA[
      <?xml version="1.0" encoding="UTF-8"?>
      <Account xmlns="http://api.rebit.org.in/FISchema/deposit" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:schemaLocation="http://api.rebit.org.in/FISchema/deposit ../FISchema/deposit.xsd" id="accountNumber" type="deposit">
    <Summary type="current" openingDate="2002-09-24" ifscCode="AIRP0000001" micrCode="" branch="" facility="OD" currentBalance=""    currentODLimit="">
          <Holders type="single">
    <Holder order="1|2" name="" aadhaar="123456789012" mobile="" landline="" address="" email="qw.1234@co.in" pan="AAAPL1234C"/></Holders>
    </Summary>
    <Transactions startDate="2002-09-24" endDate="2002-09-24">
          <Transaction id="" date="2002-09-24" narration="" reference="" amount="800" balance="90"/>
  </Transactions>
      </Account>
    ]]>
            </accountData>
        </UserAccount>
</UserAccountTrans>

    Response Expected: 

    User account transaction added.
